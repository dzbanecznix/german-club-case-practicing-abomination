# Installation
Installieren Sie erst [Python](https://www.python.org/) und fügen sie es zu PATH hinzu.
Dann müssen Sie auch ```pyttsx3``` durch ```pip install pyttsx3``` herunterladen
# Übrigens
Ich hab das sehr schnell gemacht (um nicht zu sagen, dass ich kein Informatiker oder so bin), also kann man hier viele Dinge sicherlich berbessern.<br>
Zum Beispiel glaub ich dass es gut wäre, falls man eine pre-programmierte Bibliothek finden könnte, die verschiedene Deutsche wörter delkinieren kann.<br>
